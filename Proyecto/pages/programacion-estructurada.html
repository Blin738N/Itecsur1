¿Qué es la Programación Estructurada?

La programación estructurada es un paradigma de programación que se basa en el uso de estructuras de control claras y bien definidas, como secuencias, selecciones (condicionales) y repeticiones (bucles). Este enfoque mejora la legibilidad, el mantenimiento y la eficiencia del código.

Variables y Tipos de Datos

Las variables son espacios de memoria que almacenan valores que pueden cambiar durante la ejecución de un programa. Cada variable tiene un tipo de dato que define el tipo de información que puede almacenar, como números enteros, decimales, cadenas de texto, entre otros.

int edad = 25; // Variable de tipo entero
double precio = 19.99; // Variable de tipo decimal
char letra = 'A'; // Variable de tipo carácter
String nombre = "Juan"; // Variable de tipo cadena de texto

En los ejemplos anteriores, se declaran variables de diferentes tipos de datos: int (entero), double (decimal), char (carácter) y String (cadena de texto).

Estructuras de Control

Las estructuras de control permiten definir el flujo de ejecución de un programa. Las más comunes son:

  • Secuencias: Ejecución lineal de instrucciones.
  • Condicionales: Permiten tomar decisiones basadas en condiciones.
  • Bucles: Repiten un bloque de código mientras se cumpla una condición.

Condicionales

Un condicional permite ejecutar un bloque de código si se cumple una condición. Por ejemplo:

int edad = 18;
if (edad >= 18) {
    System.out.println("Eres mayor de edad.");
} else {
    System.out.println("Eres menor de edad.");
}

Bucles

Un bucle permite repetir un bloque de código mientras se cumpla una condición. Por ejemplo:

for (int i = 1; i <= 5; i++) {
    System.out.println("Iteración: " + i);
}

Este bucle imprime los números del 1 al 5.
